# BDR Money App

Локальное веб-приложение для личного БДР: доходы, расходы, текущий месяц, лимиты, кредиты, ипотека и импорт банковских файлов.

## Запуск

Откройте приложение через GitHub Pages или локально через `index.html`.

Основные файлы:

- `index.html` — оболочка приложения.
- `app.js` — основная логика БДР.
- `config.js` — базовые настройки.
- `rules.js` — вспомогательные правила.
- `operations.js` и `operations.json` — стартовые данные или резерв.
- `CHANGELOG.md` — история изменений.

## Что умеет текущая версия

- Импорт XLSX, CSV, JSON и JS.
- Одновременная загрузка нескольких банковских файлов.
- Предпросмотр импорта перед применением.
- Дедубликация операций.
- Сохранение данных в браузере.
- Онбординг для первого запуска без операций без дублирования кнопок импорта.
- Единая рабочая вкладка операций: фильтры, поиск, ручное добавление, inline-редактирование и разделение одной транзакции на несколько категорий.
- Финансовый health score, нагрузка расходов на стабильный доход и подсказки по главной категории расходов.
- Настраиваемая месячная цель резерва, которая влияет на расчёт суммы к сокращению.
- Подключение к backend API для банковской синхронизации (`POST /api/bank/sync`) и ИИ-категоризации (`POST /api/ai/categorize`) без хранения банковских/OpenAI ключей в браузере.
- Разделение обычных доходов, стабильных доходов, ипотечных поступлений, повседневной рутины, обязательных платежей, крупных расходов, переводов, возвратов и внутренних переводов.
- Вкладка «Кредиты» с несколькими кредитами, ипотечным калькулятором и досрочными погашениями.
- Вкладка «Управление» для импорта, экспорта, лимитов, категорий и настроек.

## Важно

Приложение не считает все поступления доходами, а все списания расходами. Логика БДР разделяет операции по смыслу и исключает технические операции, внутренние переводы, ипотечные справочные поступления и пополнение наличными из обычных доходов.
