# AI‑почтовый клиент

Этот репозиторий содержит упрощённый пример почтового клиента с интеграцией искусственного интеллекта. 
Приложение подключается к почтовому серверу через IMAP, отображает список входящих сообщений и предлагает 
рекомендованные ответы с помощью LLM (например, OpenAI). Также демонстрируются возможности работы с 
вложениями, включая чтение PDF, Word и Excel, хотя в данном примере извлечение вложений не реализовано.

## Возможности

- Периодическая проверка почтового ящика через IMAP. 
- Отображение списка новых писем в веб‑интерфейсе. 
- Уведомления о новых письмах в реальном времени через WebSocket (Socket.IO). 
- Запрос к языковой модели для генерации рекомендованного ответа на выбранное письмо. 
- Простая HTML‑страница для взаимодействия.

## Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте файл `.env` в каталоге `email_ai_app` со следующими параметрами:
   ```env
   IMAP_HOST=imap.example.com
   IMAP_PORT=993
   EMAIL_USER=your_login
   EMAIL_PASSWORD=your_password
   OPENAI_API_KEY=sk-.... (опционально)
   ```
3. Запустите приложение:
   ```bash
   python app.py
   ```
4. Перейдите на `http://localhost:5000` в браузере, чтобы просмотреть интерфейс.

## Доработки

- Добавьте авторизацию SMTP для отправки исходящих писем. 
- Расширьте обработку вложений, используя библиотеки `pdfminer.six`, `python-docx` и `openpyxl` для
  извлечения текста из PDF, DOCX и Excel. Полученный текст можно передавать языковой модели для создания
  отчётов или занесения в базу данных. 
- Реализуйте сохранение писем и мета‑данных в базе данных (например, PostgreSQL), чтобы обеспечить
  долговременное хранение и быстрый поиск. 
- Подключите локальную модель (например, Llama, Mistral) для автономной работы без отправки данных
  сторонним сервисам. 
- Создайте полноценный клиентский интерфейс на React или Vue, если планируется серьёзное использование.
